# VITyarthi-project-Library-Management-System
A minimal command-line Library Management System implemented in Python

**Project Structure**
buildyourownproject/
  ├─ README.md
  ├─ statement.md
  ├─ src/
  │  └─ library_system.py
  └─ samples/
     └─ sample_run.txt
  
**Features**
Add and list books (title, author, copies)
Search books by title/author
Add borrowers
Issue books to borrowers
Return books
Reports: available books, issued books, borrower history
